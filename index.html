2) Write a program to find the count of palindromes in the given array.

input:[22,122,202,138,2]
output:2

-----------code------------

a=[22,122,202,138,2]
c=0;
for( i of a){
    emp="";
    for(j=String(i).length-1;j>=0;j--){
        emp=emp+String(i)[j]
    }
    if(emp==i){
        c=c+1;
    }
   } console.log("No of palindromes in given array are "+c)

   -----output-----
   No of palindromes in given array are 3
